Name
	sccp get repeatreltimerdelay

SYNOPSIS
	sccp get repeatreltimerdelay stackname <stack-name>

DESCRIPTION
	Gets the value for repeatreltimerdelay property.
	
PARAMETERS

	Optional Parameters

	<stack-name>	-	Name of the stack on which this command is executed. 
					If not passed, the first stack configured in ShellExecutor
					will be used.	

EXAMPLES
	sccp get repeatreltimerdelay
