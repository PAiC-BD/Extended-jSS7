Name
	sccp get cc_algo 

SYNOPSIS
	sccp get cc_algo stackname <stack-name>

DESCRIPTION
	Gets the value of sccp congestion control algorithm (international or levelDepended).

PARAMETERS

	Optional Parameters

	<stack-name>	-	Name of the stack on which this command is executed. 
					If not passed, the first stack configured in ShellExecutor
					will be used.	

EXAMPLES
	sccp get cc_algo
