Name
	sccp get periodoflogging 

SYNOPSIS
	sccp get periodoflogging stackname <stack-name>

DESCRIPTION
	Gets period of logging warning messages.
	
PARAMETERS

	Optional Parameters

	<stack-name>	-	Name of the stack on which this command is executed. 
					If not passed, the first stack configured in ShellExecutor
					will be used.	

EXAMPLES
	sccp get periodoflogging
