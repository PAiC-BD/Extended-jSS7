Name
	sccp get maxdatamessage 

SYNOPSIS
	sccp get maxdatamessage stackname <stack-name>

DESCRIPTION
	Gets Max available SCCP message data for all message types.
	
PARAMETERS

	Optional Parameters

	<stack-name>	-	Name of the stack on which this command is executed. 
					If not passed, the first stack configured in ShellExecutor
					will be used.	

EXAMPLES
	sccp get maxdatamessage
