Name
	sccp get timerexecutors_threadcount

SYNOPSIS
	sccp get timerexecutors_threadcount stackname <stack-name>

DESCRIPTION
	Gets the value for timerexecutors_threadcount property.
	
PARAMETERS

	Optional Parameters

	<stack-name>	-	Name of the stack on which this command is executed. 
					If not passed, the first stack configured in ShellExecutor
					will be used.	

EXAMPLES
	sccp get timerexecutors_threadcount
