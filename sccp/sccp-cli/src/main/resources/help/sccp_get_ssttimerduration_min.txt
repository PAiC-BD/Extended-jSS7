Name
	sccp get ssttimerduration_min 

SYNOPSIS
	sccp get ssttimerduration_min stackname <stack-name>

DESCRIPTION
	Gets the value for ssttimerduration_min property.
	
PARAMETERS

	Optional Parameters

	<stack-name>	-	Name of the stack on which this command is executed. 
					If not passed, the first stack configured in ShellExecutor
					will be used.	

EXAMPLES
	sccp get ssttimerduration_min
