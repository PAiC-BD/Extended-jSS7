Name
	sccp get respectpc 

SYNOPSIS
	sccp get respectpc stackname <stack-name>

DESCRIPTION
	Gets the value for respectpc property.
	
PARAMETERS

	Optional Parameters

	<stack-name>	-	Name of the stack on which this command is executed. 
					If not passed, the first stack configured in ShellExecutor
					will be used.	

EXAMPLES
	sccp get respectpc
