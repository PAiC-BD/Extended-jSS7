Name
	sccp get ssttimerduration_increasefactor 

SYNOPSIS
	sccp get ssttimerduration_increasefactor stackname <stack-name>

DESCRIPTION
	Gets the value for ssttimerduration_increasefactor property.
	
PARAMETERS

	Optional Parameters

	<stack-name>	-	Name of the stack on which this command is executed. 
					If not passed, the first stack configured in ShellExecutor
					will be used.	

EXAMPLES
	sccp get ssttimerduration_increasefactor
