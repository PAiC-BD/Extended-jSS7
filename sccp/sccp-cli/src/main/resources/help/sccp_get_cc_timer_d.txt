Name
	sccp get cc_timer_d 

SYNOPSIS
	sccp get cc_timer_d stackname <stack-name>

DESCRIPTION
	Gets the value of Timer A value (in milliseconds) for SCCP congestion control.

PARAMETERS

	Optional Parameters

	<stack-name>	-	Name of the stack on which this command is executed. 
					If not passed, the first stack configured in ShellExecutor
					will be used.	

EXAMPLES
	sccp get cc_timer_d
