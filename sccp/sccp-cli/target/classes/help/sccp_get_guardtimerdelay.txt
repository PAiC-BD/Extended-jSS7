Name
	sccp get guardtimerdelay

SYNOPSIS
	sccp get guardtimerdelay stackname <stack-name>

DESCRIPTION
	Gets the value for guardtimerdelay property.
	
PARAMETERS

	Optional Parameters

	<stack-name>	-	Name of the stack on which this command is executed. 
					If not passed, the first stack configured in ShellExecutor
					will be used.	

EXAMPLES
	sccp get guardtimerdelay
