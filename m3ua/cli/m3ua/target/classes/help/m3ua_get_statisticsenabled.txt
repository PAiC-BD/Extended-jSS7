Name
	m3ua get statisticsenabled 

SYNOPSIS
	m3ua get statisticsenabled stackname <stack-name>

DESCRIPTION
	Returns is statistic enabled
	
PARAMETERS

	Optional Parameters

	<stack-name>	-	Name of the stack on which this command is executed. 
					If not passed, the first stack configured in ShellExecutor
					will be used.		

EXAMPLES
	m3ua get statisticsenabled
