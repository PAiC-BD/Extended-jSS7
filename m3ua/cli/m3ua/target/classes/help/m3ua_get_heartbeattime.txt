Name
	m3ua get heartbeattime 

SYNOPSIS
	m3ua get heartbeattime stackname <stack-name>

DESCRIPTION
	Returns the Heart Beat value set
	
PARAMETERS

	Optional Parameters

	<stack-name>	-	Name of the stack on which this command is executed. 
					If not passed, the first stack configured in ShellExecutor
					will be used.		

EXAMPLES
	m3ua get heartbeattime
