Name
	tcap get backtonormalmemorythreshold_3 

SYNOPSIS
	tcap get backtonormalmemorythreshold_3 stackname <stack-name>

DESCRIPTION
	Sets MemoryMonitor Thresholds: a percent of occupied memory after which
	MemoryMonitor resumes to the congestion level 2.

PARAMETERS

	Optional Parameters

	<stack-name>	-	Name of the stack on which this command is executed. 
					If not passed, the first stack configured in ShellExecutor
					will be used.	

EXAMPLES
	tcap get backtonormalmemorythreshold_3
