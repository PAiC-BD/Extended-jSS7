Name
	tcap get swaptcapidbytes 

SYNOPSIS
	tcap get swaptcapidbytes stackname <stack-name>

DESCRIPTION
	Gets value for swaptcapidbytes. 
	
PARAMETERS

	Optional Parameters

	<stack-name>	-	Name of the stack on which this command is executed. 
					If not passed, the first stack configured in ShellExecutor
					will be used.	

EXAMPLES
	tcap get swaptcapidbytes
