Name
	tcap get dialogidletimeout 

SYNOPSIS
	tcap get dialogidletimeout stackname <stack-name>

DESCRIPTION
	Gets value for dialog timeout. 
	
PARAMETERS

	Optional Parameters

	<stack-name>	-	Name of the stack on which this command is executed. 
					If not passed, the first stack configured in ShellExecutor
					will be used.	

EXAMPLES
	tcap get dialogidletimeout
